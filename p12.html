Introducere in AJAX



Descriere: Cuprins: Prezentare generala a tehnicii AJAX • Introducere in JavaScript • Nucleul limbajului JavaScript • Accesarea arborelui DOM in browser • Obiectul XMLHttpRequest • Sabloane de interactiune Web bazate pe AJAX • Introducere in JSON • Accesarea serviciilor Web folosind AJAX • Securitatea in JavaScript • Principii de proiectare a aplicatiilor Web bazate pe AJAX Descriere: AJAX (Asynchronous JavaScript and XML) reprezinta o tehnica bazata pe comunicarea asincrona dintre clientul si serverul Web, prin care o serie de tehnologii sint utilizate impreuna in scopul maririi interactivitatii aplicatiilor Web. Lucrarea ofera cititorilor cu un nivel mediu de pregatire in domeniul programarii Web o initiere completa in AJAX. Sint prezentate nucleul limbajului JavaScript, utilizat pentru procesarea datelor, formatul JSON (JavaScript Object Notation), folosit alaturi de XML (Extensible Markup Language) pentru schimbul de date, precum si obiectul XMLHttpRequest, utilizat in transferul asincron al datelor. O alta tema abordata este cea a accesarii arborilor DOM (Document Object Model) XHTML si DOM XML in browser. De asemenea, sint analizate si exemplificate o serie de sabloane de interactiune Web si instrumente bazate pe AJAX.
